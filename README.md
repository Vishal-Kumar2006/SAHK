🍽️ SAHK – Swad Apne Haath Ka

A MERN Stack Recipe-Sharing Platform for Food Lovers

🚀 SAHK (Swad Apne Haath Ka) is a full-stack web application where food lovers can explore, share, and discover recipes based on meal types and cooking methods. It's a growing platform built with the MERN stack (MongoDB, Express.js, React.js, Node.js) and designed to celebrate the art of cooking with love. ❤️

🚀 Getting Started (Local Setup)

Clone the Repository

git clone https://github.com/Vishal-Kumar2006/SAHK.git  
cd SAHK  
Install Dependencies

For server:

cd backend  
npm install  

For client:

cd ../frontend  
npm install  

Start Development Server

# In backend folder  
npm run start  

# In frontend folder  
npm run dev  
Open in your browser: http://localhost:3000
